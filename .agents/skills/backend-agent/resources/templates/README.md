# Backend Agent - API Endpoint Templates

Full CRUD endpoint templates per stack. Use as a starting point and adapt.

| File | Stack |
|---|---|
| `python.md` | FastAPI full CRUD router + service |
| `typescript.md` | Express full CRUD router + service |
| `go.md` | Echo full CRUD handler + service |

These templates demonstrate:
- List (paginated + filtered)
- Get single (with auth check)
- Create
- Update (partial)
- Delete (soft + hard)
- Bulk create
